# uDOS-empire Overview

`uDOS-empire` is the family module for outbound operations, publishing, contact
workflows, and reusable operational packs.

It is not the host runtime, not the family contract layer, and not the browser
surface. It is the place where inspectable, operator-facing workflows are
assembled and run.

## Empire Owns

- publishing systems
- campaign and messaging workflows
- contact and audience operations
- reusable packs, scripts, mappings, and templates
- educational make-and-deploy patterns

## Empire Should Feel Like

- a practical operations workshop
- dry-run first
- approval-aware for high-impact actions
- inspectable before live writes
- remixable for learning and adaptation
