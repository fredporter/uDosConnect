# Unit 01: Empire Basics

## What This Module Is

`uDOS-empire` is the family WordPress-plugin module for local CRM, contact,
email, and admin operations running on `uDOS-host`.

## What You Should Learn

By the end of this unit you should be able to:

- explain what Empire owns
- run the basic repo checks
- identify the WordPress-plugin boundary
- describe why Ubuntu owns Git or GitHub host actions instead

## Practical How-To

1. Run the repo checks.
2. Read the plugin architecture note.
3. Read the plugin data model.

```bash
bash scripts/run-empire-checks.sh
```

## Quick Check

You pass this unit if you can answer:

- What does Empire own?
- Why is Empire now WordPress-centred?
- Why does Ubuntu own host-side Git or GitHub actions?
