# Legacy Pack Inventory

`packs/legacy/` documents packs that still exist for compatibility and
transition coverage while Empire moves toward the WordPress-plugin model.

The current legacy-transition packs are:

- `campaign-starter`
- `contact-import-cleanup`
- `event-launch`
- `launcher-installer`
- `quickstart`
- `weekly-bulletin`

The current active pack is:

- `wordpress-contact-import`

Rule:

- new default examples should prefer current packs
- legacy-transition packs may stay runnable until their runtime seams are
  retired or replaced
