# Config

`config/` is the checked-in configuration lane for `uDOS-grid`.

Family placement rule:

- put checked-in non-secret spatial config templates and config examples here
- keep starter place, artifact, and layer payloads in `examples/` or `seed/`
- reserve `defaults/` for reusable baseline profiles only if the repo later
  needs them

Current state:

- intentionally minimal during activation
- reserved for spatial projection and index config examples that belong to Grid
