# Unit 01: Grid Basics

## What This Module Is

`uDOS-grid` is the family module for canonical spatial identity, layers, place
references, and place-bound artifact lookup.

## What You Should Learn

By the end of this unit you should be able to:

- explain what Grid owns
- locate the checked-in contracts and seed data
- run the repo checks
- describe the difference between spatial truth and renderer truth

## Practical How-To

1. Read the contracts lane.
2. Inspect the seed registry.
3. Run the repo checks.

```bash
cat contracts/README.md
cat seed/README.md
bash scripts/run-grid-checks.sh
```

## Editable Demo
