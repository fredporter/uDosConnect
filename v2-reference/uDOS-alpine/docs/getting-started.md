# uDOS-alpine Getting Started

Use this order:

1. Read `docs/README.md`.
2. Read `docs/activation.md`.
3. Read `docs/alpine-runtime-profile.md`.
4. Review `profiles/` and `distribution/`.
5. Run `bash scripts/run-alpine-checks.sh`.

Then move into the lane that matches the work:

- package definitions: `apkbuild/`
- service assumptions: `openrc/`
- release rules: `docs/release-policy.md`
- ThinUI handoff: `docs/thin-gui-launcher-integration.md`

Add deployment examples only when they match supported profiles.
