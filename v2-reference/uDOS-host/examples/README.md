# Examples

Starter bootstrap payloads for autoinstall, first-run, and command-centre
scaffolding.

- `command-centre-demo/` — static HTML served by `scripts/serve-command-centre-demo.sh` for lane 1 browser proof.
- `thinui-c64-launch.json` — demo ThinUI seed (C64 theme).
- `thinui-ubuntu-gnome-launch.json` — ThinUI seed aligned with **uDOS-surface** `ubuntu-gnome` profile (`udos-default` / windowed).
