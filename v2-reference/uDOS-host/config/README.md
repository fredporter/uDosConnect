# Config Lane

Checked-in non-secret base image defaults for packages, users, services, and
system-level settings.

Planned command-centre config structure is documented in:

- `../docs/config-layout.md`
- `../docs/systemd-unit-plan.md`

Active checked-in config lanes now include:

- `config/env/`
- `config/runtime/`
- `config/policy/`
- `config/systemd/`
