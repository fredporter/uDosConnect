# Source

This directory holds the canonical checked-in source artifacts for
`uDOS-groovebox`.

- `pattern-library.json` defines the current public pattern-library shape
- `songscribe-bridge.json` defines the current text-to-pattern bridge shape
- `sample-bank.json` defines the local sample and drum-bank lane
- `synth-presets.json` defines the local synth preset lane
- `songscribe-request.json` defines the bridge request contract
