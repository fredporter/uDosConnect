# v2.0.1 Theme Foundation

The theme repo owns the shared starter token maps for shell and publishing
surfaces in `v2.0.1`.

## Starter Surfaces

- core theme tokens in `src/theme-tokens.json`
- shell-facing mappings in `src/shell-theme-map.json`
- publishing and email-safe mappings in `src/publishing-theme-map.json`

## Scope

This round establishes the token bridge only. Product-specific visual systems
and richer theme packs belong to later rounds.
