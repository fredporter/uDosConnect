# v2.0.1 Theme Foundation

This document records the baseline theme ownership and adapter split for the
public `uDOS-themes` repo.

## Purpose

Define the stable foundation that later rounds build on:

- token ownership lives in `uDOS-themes`
- runtime repos consume theme identities and adapter contracts
- appearance stays separate from runtime semantics

## Foundation Surfaces

- `src/theme-tokens.json`
- `src/shell-theme-map.json`
- `src/publishing-theme-map.json`
- `src/adapters/thinui/`

## Boundary

- `uDOS-core` owns semantics
- `uDOS-shell`, `uDOS-thinui`, and `uDOS-wizard` consume theme contracts
- `uDOS-themes` owns render tokens, fonts, loaders, and adapter mappings

## Current Extension

The current ThinUI adapter lane now includes:

- `thinui-c64`
- `thinui-nes-sonic`
- `thinui-teletext`
- `minimal-safe`
