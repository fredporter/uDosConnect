# uDOS-themes Boundary

## Owns

- public themes
- visual tokens
- component primitives
- reusable asset packs
- surface adapters
- prose presets
- publishing and email-safe theme adapters
- workflow and GTX-form theme adapters
- gameplay and product skin theme adapters

## Does Not Own

- shell interaction behavior
- runtime contracts
- private product visual identity
- final UI behavior semantics inside consuming runtimes
