# Unit 01: Themes Basics

## What This Module Is

`uDOS-themes` is the family module for public themes, tokens, adapters, skins,
and cross-surface visual primitives.

## What You Should Learn

By the end of this unit you should be able to:

- explain what Themes owns
- describe the difference between themes, adapters, and runtimes
- identify the main surface lanes covered here

## Practical How-To

1. Read the activation doc.
2. Read the architecture doc.
3. Inspect the token and adapter areas under `src/`.

## Quick Check

You pass this unit if you can answer:

- What do themes define?
- What do adapters do?
- What should remain outside this repo?
