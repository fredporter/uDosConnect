# v2.1 Core API And MCP Quickstart

This guide gives a runnable path for the core MCP contract lane.

## Contract Surfaces

- contracts/mcp-tool-contract.json
- schemas/mcp-tool-contract.schema.json
- docs/MCP-SCHEMA.md

## Validate Contract Tests

```bash
bash scripts/run-core-checks.sh
python3 -m pytest tests/test_ok_agent_contracts.py -q
```

## Validate Schema Structure Quickly

```bash
python3 - <<'PY'
import json
from pathlib import Path

contract = json.loads(Path("contracts/mcp-tool-contract.json").read_text())
schema = json.loads(Path("schemas/mcp-tool-contract.schema.json").read_text())

print("contract keys:", sorted(contract.keys())[:8])
print("schema title:", schema.get("title"))
PY
```

## Boundary Rule

Core defines contract semantics only. Transport and managed routing remain outside core.
