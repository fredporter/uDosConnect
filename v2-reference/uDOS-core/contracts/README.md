# Contracts

Canonical command, workflow, plugin, container, runtime, render, automation,
and OK Agent contract surfaces.

## Vault Survival Contracts (v2.0.8)

- `vault-survival.md` — canonical vault crash survival protocol: crash-safe/discard-safe boundary, `.compost` format, progressive file versioning, sandbox draft/backup/restore guards, snap-off portable layer, and reconnect handshake
- `vault-survival-contract.json` — machine-readable schema for `.compost` header/surface/footer records, delta records, sandbox operation descriptors, and vault state boundary descriptors

## Spatial Contracts (v2.0.5)

## Language and Script Contracts (v2.0.4)
