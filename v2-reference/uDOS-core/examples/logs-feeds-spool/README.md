# Logs, Feeds, and Spool Examples

These examples demonstrate the stable event-layer contracts defined by
`uDOS-core`.

Included:

- `event-record.example.json`
- `feed-item.example.json`
- `feed-create-request.example.json`
- `spool.example.json`

Use these examples for:

- contract validation fixtures
- documentation snippets
- test fixtures for downstream repos
- implementation guidance for MCP and UI consumers
