# Shell Docs

`/docs` is the stable reference lane for `uDOS-shell`.

Use it for:

- shell purpose and ownership
- activation and local setup
- terminal interaction contracts
- examples and operator-facing how-tos
- shell consumption of family contracts

Use `/wiki` for beginner-friendly learning units, quick practical guides, and
simple demos.
