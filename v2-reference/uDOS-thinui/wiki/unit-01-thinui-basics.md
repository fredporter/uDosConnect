# Unit 01: ThinUI Basics

## What This Module Is

`uDOS-thinui` is the family module for low-resource fullscreen GUI runtime
surfaces that sit between the TUI and a full browser app.

## What You Should Learn

By the end of this unit you should be able to:

- explain what ThinUI owns
- describe the difference between ThinUI and browser presentation
- identify the main runtime scaffold pieces

## Practical How-To

1. Read the ThinUI spec.
2. Read the runtime loop scaffold.
3. Run the local checks.

```bash
bash scripts/run-thinui-checks.sh
```
