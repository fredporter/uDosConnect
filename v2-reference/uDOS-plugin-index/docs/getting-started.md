# uDOS-plugin-index Getting Started

1. Read `docs/README.md`.
2. Start with the manifest contracts in `contracts/`.
3. Review `docs/activation.md`.
4. Align validation in `schemas/`.
5. Run `bash scripts/run-plugin-index-checks.sh`.
6. Add examples only when they teach a stable contract.
7. Keep tests focused on schema and manifest integrity.
