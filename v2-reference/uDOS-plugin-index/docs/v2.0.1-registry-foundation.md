# Registry Foundation

The plugin index owns the starter metadata model for discovery and trust.

## Starter Surfaces

- plugin manifest schema with trust and compatibility metadata
- certification placeholders for future promotion logic
- wrapped source metadata for GitHub, package, or local pathways
- starter catalog entries in `examples/catalog.json`

## Trust Classes

- `certified`
- `community`
- `local`

## Scope

This round defines metadata only. Install resolution and runtime execution stay
with `uDOS-core` and the owning repos.
