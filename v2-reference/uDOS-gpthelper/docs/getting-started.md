# Getting started

1. Read **`README.md`**, **`docs/agent-digital-v2_1-spec.md`**, and **`docs/action-schema.md`**.
2. For **Agent Digital**: apply patches under **`prompts/patches/`** and **`docs/agent-digital-export-integration.md`**.
3. For **uDOS Developer**: read **`docs/udos-developer-export-integration.md`**.
4. For local ZIP export, deploy **`uDOS-host`** `services/gpt-export-helper` and follow **`uDOS-host/services/gpt-export-helper/README.md`**.
5. Run checks: **`bash scripts/run-gpthelper-checks.sh`**.
