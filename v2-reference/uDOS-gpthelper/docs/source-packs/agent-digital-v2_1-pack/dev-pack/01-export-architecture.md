# Export Architecture v2.1
## Export Priority
1. ZIP file
2. Canvas multi-file
3. File-by-file (one file per reply)
4. Single Binder doc (only for non-code)
## File-by-file Rules
- One file per reply
- Include file number and total
- Include full path
- Include file type
- Include save instructions
- End with: reply NEXT
## Why
Prevents confusion when saving multiple files manually
Ensures consistency across all outputs
