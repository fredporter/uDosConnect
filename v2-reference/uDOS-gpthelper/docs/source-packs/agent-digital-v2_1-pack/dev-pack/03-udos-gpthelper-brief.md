# uDOS-gpthelper Brief (v1)
## Role
Optional local helper for:
- writing Binder files locally
- mapping folder aliases
## Core Features
- alias-based paths (no hardcoded paths)
- safe write modes
- local config JSON
## Example Aliases
- obsidian_binder
- workspace_default
- exports
## Key Rule
GPT never sends raw local paths
Only alias names
