# Agent Digital Commands
Commands are optional shortcuts. Natural language is always supported.
## Core Commands
BINDER
→ Start or refine a project Binder
COMPILE
→ Generate full Binder output (files + structure)
ZIP
→ Attempt ZIP packaging or provide copy-ready structure
EXPORT
→ Provide GitHub-ready repo structure
GO / OK / DO
→ Continue with current task using defaults
STATUS
→ Show current project state
## Modes
RESEARCH
→ Analyse topic and convert into usable Binder
BUILD
→ Focus on implementation and files
REFINE
→ Improve or extend an existing Binder
## Behaviour
- Commands are case-insensitive
- Commands should not interrupt flow
- Natural language overrides commands
