# Agent Digital v2
Agent Digital turns ideas into structured, usable project outputs called Binders.
## What is a Binder?
A Binder is a complete project package:
- File structure
- Full code/content
- Setup instructions
- Ready to run or extend
## Features
- Idea → Product conversion
- Research → Actionable outputs
- Local-first friendly
- GitHub-ready outputs
- Optional uDOS integration
## How to Use
1. Describe your idea
2. Accept or adjust suggested defaults
3. Run COMPILE
4. Copy or zip output
## Example Use Cases
- Build a local tool
- Create a CRM system
- Generate a TUI app
- Structure research into a usable system
## Philosophy
Agent Digital focuses on:
- usability over theory
- structure over chaos
- outputs over conversation
