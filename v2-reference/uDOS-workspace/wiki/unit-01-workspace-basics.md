# Unit 01: Workspace Basics

## What This Module Is

`uDOS-workspace` is the binder-first visual operator shell for browser-based
workspace surfaces.

## What You Should Learn

By the end of this unit you should be able to:

- explain what Workspace owns
- describe the difference between operator truth and canonical truth
- identify the main browser surfaces

## Practical How-To

1. Read the workspace activation doc.
2. Read the workspace architecture note.
3. Inspect the wireframe and compiler manifest docs.

## Quick Check

You pass this unit if you can answer:

- What does Workspace own?
- Which repo owns canonical binder and spatial contracts?
- Which surfaces are in the first workspace shell?
