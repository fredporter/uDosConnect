# Source

`src/` is the gameplay module lane for `uDOS-gameplay`.

Current source surfaces include:

- `gameplay-state.json` as the smallest checked-in gameplay state contract
- `grid-consumption.json` as the Grid-consumption boundary note

Boundary rule:

- keep gameplay-facing state contracts here
- keep canonical semantics in `uDOS-core`
- keep canonical spatial truth in `uDOS-grid`
