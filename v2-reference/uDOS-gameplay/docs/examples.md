# uDOS-gameplay Examples

Current starter examples:

- `basic-gameplay-state.json`
- `basic-grid-gameplay-state.json`
- `google-mvp-crypt-placement-world.json`

Supporting contract note:

- `docs/google-mvp-world-contract.md`

The Grid-backed example shows how Gameplay can consume canonical place and
artifact truth from `uDOS-grid` without taking over spatial persistence.

The Google MVP example shows how a generated multiplayer world must be exported
back into a gameplay-owned artifact instead of treating cloud state as
canonical.
