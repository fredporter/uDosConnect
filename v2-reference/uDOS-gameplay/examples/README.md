# Examples

`examples/` is the smallest inspectable walkthrough lane for
`uDOS-gameplay`.

Current example lanes:

- `basic-gameplay-state.json` for the smallest gameplay state contract example
- `basic-grid-gameplay-state.json` for Grid-backed place and artifact context
- `google-mvp-crypt-placement-world.json` for the first Google MVP exportable
  world-state example

Supporting contract note:

- `../docs/google-mvp-world-contract.md`

Phase 1 rule:

- keep examples aligned to public gameplay surfaces
- keep runtime execution behavior in the owning repos
- keep generated world output exportable into repo-owned example files
