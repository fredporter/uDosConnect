# uHOME Product Split
uHOME-client = kiosk, controller, local-network UI surfaces
uHOME-server = downstream uHOME service stream, local household services,
local launcher and presentation helpers
uDOS-host = Linux-based command-centre runtime, vault host, Beacon Activate
content surfaces, persistent services
uDOS-empire = always-on Google or HubSpot sync, webhook-style server actions,
console CRM and workflow UI, HubSpot sync for app-owned contact records
