# General Knowledge Library

Survival-guide and themed how-to articles; depth, research standards, exemplars;
and **boundary** pieces that state what uDOS may recommend (e.g. uCode journeys,
compatible devices) versus out-of-family or greenfield detours.

Normative framing: `architecture/16_general_knowledge_library.md`.

**Drop article files here** (this directory), flat or in subfolders by topic.
This README is the **index**; list new entries below as you add them.

## Contributing

- **Improvements** — same as the rest of `uDOS-docs`: PRs with clear scope,
  review, and merge. Treat it like wiki-style enhancement: corrections, new
  articles, better links.
- **Bundled copy** — snapshots for offline or Core/docs checks are **read-only**
  copies of a tagged release; they are not edited in place on disk.
