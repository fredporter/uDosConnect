## Intent

Classic Modern is the shared design system and UX philosophy for a de-modernised Ubuntu host and a distraction-free uDOS primary surface.

- Ubuntu remains the host/runtime posture.
- ThinUI becomes the preferred fullscreen work surface.
- Shell and utility lanes align visually and behaviorally.
- Sonic Screwdriver is **TUI-only** for install/repair/doctor.
