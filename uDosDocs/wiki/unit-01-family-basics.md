# Unit 01: Family Basics

## What This Repo Is

`uDOS-docs` is the public knowledge spine for the family. It explains how the
repos fit together and where to start.

## What You Should Learn

By the end of this unit you should be able to:

- explain what `uDOS-docs` owns
- identify where stable explanation ends and implementation repos begin
- choose a good first repo to read next

## Practical How-To

1. Read the onboarding page.
2. Read the family architecture page.
3. Pick one path: use, learn, or build.

## Quick Check

You pass this unit if you can answer:

- What is the role of `uDOS-docs`?
- Which repo should you read for implementation details?
- What is the difference between `/docs` and `/wiki`?
