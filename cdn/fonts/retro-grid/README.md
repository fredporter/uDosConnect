## Usage

```css
/* C64 Style */
.c64-text {
  font-family: 'C64_Pro_Mono', monospace;
  font-size: 14px;
  line-height: 1.2;
}

/* Teletext Style */
.teletext {
  font-family: 'Teletext50', monospace;
  font-size: 14px;
  color: #00FF00;
  background: #000000;
}
```

## Installation

```bash
# Download C64 fonts
curl -O https://www.kreativekorp.com/software/fonts/c64/C64_Pro_Mono.ttf

# Place in retro-grid directory
mv C64_Pro_Mono.ttf /cdn/fonts/retro-grid/
```
