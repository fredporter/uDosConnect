# v3 Roadmap (Current)

Status: active
Updated: 2026-04-08

## Scope

This roadmap is for the **uDOS-v3 monorepo** and its direct execution path.

Primary build target:

- `uDOS-v3` (Host + Hivemind + ThinUI + optional WordPress)

Companion UX/interchange lane:

- `UniversalSurfaceXD` v2 (design/interchange support, not runtime authority)

## Out of scope

- New v2 compatibility work
- Reopening v2 completion lanes
- Backporting v3 behavior to legacy split repos by default

## Backlog model

Backlog is tracked as a feed in `v3-feed.md` using statuses:

- `planned`
- `in_progress`
- `done`
- `deferred`

Each entry should include why it exists and links to evidence.

## v2 reference

v2 family plans, rounds, and status remain archived at:

- `@dev/notes/roadmap/archive/v2/` (roadmap ledgers and round files)
- `docs/archive/v2/` (family dev docs that accompanied those plans)

Treat v2 as historical context only: what was planned, what shipped, and what was deferred.
