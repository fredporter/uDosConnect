# `@dev/triage`

Use this lane only for short-lived cross-repo triage that still has no clear
owner.

Keep this lane near-empty. Once ownership is known, move the work into the
owning repo or distill it into a family pathway note.

Do not keep "moved" stubs here. Delete them once the owning repo has the real
record.

## Required Questions

- what is the work item?
- which repo owns the source of truth?
- is this private, public, or cross-repo?
- which downstream docs are needed?
- which parts are implementation detail vs reusable contract?
