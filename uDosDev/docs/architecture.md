# uDOS-dev Architecture

uDOS-dev is the shared development and education workspace for the public family.

## Main Areas

- `@dev/` is the intake and collaboration surface.
- `@dev/notes/roadmap/` is the canonical live roadmap surface.
- `automation/` holds repeatable workflow support.
- `courses/` and `docs/` teach maintainers and contributors.
- `scripts/run-dev-checks.sh` is the activation validation entrypoint.
- `docs/release-surfaces.md`, `docs/release-matrix.md`, and
  `docs/versioning-policy.md` define the first release-surface lane.
- `docs/release-backlog.md` tracks remaining repo-by-repo release hardening.
